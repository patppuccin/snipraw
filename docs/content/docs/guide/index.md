# Guide

This section covers everything you need to get snipraw running and configured. If you are here for the first time, follow the pages in order — installation, then configuration, then the CLI reference.

Snipraw has no daemon, no background service by default, and no config file required to get started. The fastest path from zero to running is about two minutes.

Find the pages below to get started:

- [Installation](/docs/guide/installation) — download the binary and run it
- [Configuration](/docs/guide/configuration) — config file reference
- [Command Line](/docs/guide/cli) — flags and options for the CLI
- [Keyboard Shortcuts](/docs/guide/keyboard-shortcuts) — shortcuts available on the browser UI
