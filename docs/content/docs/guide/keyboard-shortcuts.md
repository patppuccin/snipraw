# Keyboard Shortcuts

Snipraw's browser UI supports a small set of keyboard shortcuts for faster navigation.

## Navigation

| Shortcut | Action                         |
| -------- | ------------------------------ |
| `g h`    | Go to home (project list)      |
| `g r`    | Go to raw view of current file |
| `[`      | Previous file in project       |
| `]`      | Next file in project           |

## View

| Shortcut | Action                      |
| -------- | --------------------------- |
| `t`      | Toggle theme (light / dark) |

## Other

| Shortcut | Action                           |
| -------- | -------------------------------- |
| `?`      | Show keyboard shortcut reference |
| `Escape` | Dismiss overlay                  |

::: note
Shortcuts are disabled when focus is inside an input field.
:::
