# Roadmap

A rough look at what's been built and what's coming.

## Done

- Filesystem-based project and snippet browsing
- Syntax highlighting via Chroma
- Markdown rendering with GFM callouts (`[!NOTE]`, `[!TIP]`, etc.)
- Inline image rendering with relative path rewriting
- Light/dark theme toggle with FOUC prevention
- Keyboard shortcuts
- Support `RAW` content pages
- MIME-aware content serving
- Containerization support

## Planned

- Global search
- Pin/favorite snippets using browser local storage
